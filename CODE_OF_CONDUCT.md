# Code of Conduct

The dAvePi project adopts the [Contributor Covenant, version 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/)
as its code of conduct. Read the full text at the link.

In short: this project welcomes contributions from anyone, and
asks every participant — contributors, maintainers, users in
Discussions or Discord — to act respectfully toward each other in
all project spaces (the GitHub repository, Discussions, Discord,
and any official social channels).

## Reporting

If you experience or witness conduct that violates the Covenant,
report it to the maintainers at **conduct@davepi.dev**. Reports go
to the [project maintainers](./MAINTAINERS.md), are treated
confidentially, and get an acknowledgement within 3 business days.

In urgent cases — ongoing harassment, threats — the same address
is monitored more frequently; mark the subject line `URGENT`.

## Enforcement

Maintainers follow the [Covenant's enforcement ladder](https://www.contributor-covenant.org/version/2/1/code_of_conduct/#enforcement-guidelines):

1. **Correction** — private, written feedback. The default for a
   first-time misstep.
2. **Warning** — formal warning with a defined period of avoiding
   the affected parties.
3. **Temporary ban** — removal from project spaces for a set
   period.
4. **Permanent ban** — removal from all project spaces.

The maintainers make enforcement decisions. There's no
public-shaming step; the goal is to keep the project welcoming,
not to punish.

## Scope

This Code of Conduct applies inside every official project
space — the GitHub repository (issues, PRs, Discussions, commit
messages), the Discord server, and any official social
accounts — and when someone is officially representing the project
in a public forum (e.g. speaking on its behalf, using a verified
project account).
